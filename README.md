# 🤖 Google Drive Assistant

Bot que pesquisa arquivos no Google Drive e responde perguntas usando IA Gemini.

## 🚀 Funcionalidades

- Busca arquivos no Drive por nome ou conteúdo
- Usa Gemini para gerar respostas inteligentes
- Interface simples no terminal

## 📋 Pré-requisitos

- Python 3.8+
- Conta Google Cloud com Drive API ativada
- Chave da API Gemini

## 🔧 Instalação

```bash
git clone https://github.com/SEU_USUARIO/bot-drive.git
cd bot-drive
pip install -r requirements.txt
